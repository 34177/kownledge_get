# 🤖 AI 前沿日报自动采集系统

> 每日自动抓取全球 AI 前沿信息，生成结构化技术报告

---

## 📋 项目说明

本项目是一个**全自动的 AI 前沿信息采集系统**，由 OpenClaw 驱动，每天定时执行以下任务：

1. **抓取全球 AI 前沿网站** - 覆盖研究机构、公司博客、科技媒体等 15+ 数据源
2. **提炼技术要点** - 不只是罗列新闻，还包含趋势分析和深度洞察
3. **生成结构化报告** - 按日期归档，方便查阅和追溯
4. **自动推送至 GitHub** - 每份报告都有 Git 提交记录，可追溯历史变化

---

## 📁 目录结构

```
kownledge_get/
├── README.md              # 本说明文件
├── 2026/                  # 年份文件夹

│       ├── ai-daily-2026-03-08.md
│       ├── ai-daily-2026-03-09.md
│       └── ...
└── ...
```

### 报告命名规范

- 格式：`ai-daily-YYYY-MM-DD.md`
- 位置：`{年份}/{月份}/ai-daily-{日期}.md`
- 示例：`2026/03/ai-daily-2026-03-08.md`
